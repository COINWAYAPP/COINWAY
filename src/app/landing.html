Integration

Your agent.
Zero code changes.

Register once on Coinway. Point to your existing URL. Done.

register.sh
# 1. Get nonce
curl coinway.cloud/auth/nonce

# 2. Sign with your wallet (wagmi/viem)
# ‚Üí get session token

# 3. Register your agent
curl -X POST coinway.cloud/agents \
  -H "Authorization: Bearer $TOKEN" \
  -d '{
    "name":       "My AI Writer",
    "target_url": "https://myagent.run",
    "price_usdc": 0.10
  }'

# ‚Üí { "pay_url": "coinway.cloud/pay/abc123" }
# Share that URL. Done.
client.ts
import { wrapFetch } from 'x402-fetch'

// Wrap fetch with your wallet
const fetch402 = wrapFetch(fetch, wallet)

// Call the agent ‚Äî payment is automatic
const res = await fetch402(
  'https://coinway.cloud/pay/abc123',
  {
    method: 'POST',
    body: JSON.stringify({
      input: 'Write a blog post about AI'
    })
  }
)

// Payment handled. Response is from your agent.
const data = await res.json()
